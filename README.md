# 🚀 N8N Workflows Collection

A collection of useful and powerful **n8n workflows** for AI automation, Telegram bots, email handling, Google Calendar integration, form processing, and news summarization.

## 📌 Overview

This repository contains ready-to-use n8n workflows that help automate repetitive tasks and build AI-powered systems with ease. Simply import the workflow into your n8n instance, configure the required credentials, and start automating.

## ✨ Features

- 🤖 AI-powered workflows
- 📧 Email automation
- 📅 Google Calendar integration
- 📱 Telegram bot workflows
- 📝 Form processing and HTTP requests
- 📰 Automated news summarization
- ⚡ Easy to import and customize

## 🛠 Prerequisites

Before using these workflows, ensure you have:

- [n8n](https://n8n.io/)
- API keys required by specific workflows
- Gmail account credentials (for email workflows)
- Telegram Bot Token (for Telegram workflows)
- Google Calendar credentials

## 📥 Installation

### Clone the repository

```bash
git clone https://github.com/Satwiksai36/N8N-Workflows.git
```

### Import a workflow

1. Open your **n8n** instance.
2. Click **Import Workflow**.
3. Select the desired JSON workflow file.
4. Configure the required credentials.
5. Activate the workflow.
